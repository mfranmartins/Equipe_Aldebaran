# Objetivo do projeto
O objetivo principal deste projeto consiste no desenvolvimento e simulação de um sistema modular de segurança de dados para uma cabine espacial, unificando duas camadas complementares de inteligência computacional: 

Camada de Hardware Booleano Crítico: Uma malha baseada em lógica combinacional pura, implementada por Circuitos Integrados (CIs) de portas lógicas discretas (AND, OR, NOT), responsável por computar instantaneamente o status de colapso da nave (X = 1). 

Camada de Software Preditivo (IA): Um algoritmo assíncrono executado em um microcontrolador que analisa tendências dinâmicas ("Rampas" de subida ou aproximação de zonas perigosas) e avisa visualmente o operador de forma preditiva por meio de um display LCD adaptativo, antes mesmo que os limites elétricos do hardware físico sejam rompidos.
